#jotaalvim-tool

##ddl - Delete duplicated lines
Delete repeated lines in file, ignores empty lines and lines who have ``` `{ }``` in them. Similar to
```
sort | uniq
```
or
```
sort -u
```
however, using this tool you keep the same relative order in the file.
